Write a code in the blanks to search the given element in the given array and return 0 if present or -1.

Input :
The input consists of n + 2 lines. The first line consists a single integer n, The next n lines consist of 1 integer element part of the array. The last line consists of an integer to be searched.
Output found or missing based on whether the element is present in the array or not.

Constraints :
1 <= n <= element <= 100.

Output :
Print Found if present or Missing.
