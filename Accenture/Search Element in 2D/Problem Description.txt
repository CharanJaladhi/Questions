Write a code in the blanks to find the index of the given search element of the two dimensional matrix, and return the index of the element to the main.

Input :
Get a target element and return its coordinates. The first line of input is the sizeof row and column, followed rxc elements. The third line of input is the element to be searched in the rxc matrix.

Output :
If the value didn't exist, the program had to return (-1,-1), or the index of ( row col ).

Example:
Input
4 2
0 9 8 7 6 5 4 3
3

Output
3 1
