You are required to implement the following function.

         int OperationChoices(int num1, int num2 , int choice);

The function accepts 3 positive integers ‘num1’ , ‘num2’ and ‘choice ‘ as its arguments. Implement the function to return.

( num1 + num2 ) , if choice=1
( num1 + num2 ) , if choice=2
( num1 * num2 ) ,  if choice=3
(num1 / num2 ) ,  if choice=4
Assumption : All operations will result in integer output

Input 1:
choice : 1
num1 : 12
num2 : 16

Output 1:
Since ‘choice’=1 , (12+16) is performed which is equal to 28 , hence 28 is returned.
 
Input 2:
choice : 2
num1 : 16
num2 : 20

Output 2:
-4
