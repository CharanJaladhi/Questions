Secret message agency provides message encoding and decoding services for Secure data transfer. The first step in decoding includes removal of special characters from the message as special characters and white spaces do not hold any meaning.

Write an algorithm to help the agency find the number of special characters in a given message  

Input
The input consists of a string message,
representing the message that need to be decoded by the agency

Output
Print an integer representing the number of special characters present in a given message 

Example
Input
gasgg54@#vscsd!s*

Output
4

Explanation
The special characters having no meaning are [@, #, !, *]
