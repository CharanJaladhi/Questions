The cosmetic company "BeautifyMe"has decided to reward its brand promoters with a batch of cosmetics. Each batch is identified by a unique numeric barcode. Each digit of the barcode number represents a product ID of the products present in the batch.Before distributing the products to its promoters.the company wants to make sure that all products present in the batch are still fresh and safe to use. The distribution center has therefore been advised to identify the oldest products present  in each batch. The oldest product in each batch is the one with the smallest product ID. There can be multiples of the same product in one batch

Write an algorithm to identify the oldest product present in the batch. 

Input
The Input consists of an integer - barcode, representing the numeric barcode of the batch of cosmetics. 

Output
Print an integer representing the ID of the oldest product present in the batch. 

Constraint
0 < barcode <= 10^9

Example
Input 
345627 

Output
2 

Explanation
 The smallest product ID as 2. Hence it is the oldest product.
