The garments company Apparel wishes to open outlets at various locations. The company shortlisted several plots in these locations and wishes to select only plots that are square-shaped.
Write an algorithm to help Apparel find the number of plots that it can select for its outlets.

INPUT
The first line of the input consists of an integer numofplots, representing the number of plots shortlisted by the company for the outlets(N).
The second line consists of N space-separated integers-area1,area2,....,areaN representing the area of the N plots selected for outlets.

OUTPUT
Print an integer representing the number of plots that will be selected for outlets.

CONSTRAINTS
0 <= numofplots <=10^6
0 <= area <=106
0 <= i < num of plots

EXAMPLE 
Input:
8
79 77 54 81 48 34 25 16
Output:
3
Explanation:
The areas that are in square form are 81,25 and 16.
so,the output is 3.
