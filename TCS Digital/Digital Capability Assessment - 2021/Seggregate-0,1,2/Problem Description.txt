Given a list A of integer numbers that have values coming from SET={0,1,2}. The task is to put all 1s first, then all 0s, and all 2s at the last.

For Example, 
List A[]={0,1,2,0,1,2}
You have to put all 1s first, after that all 0s, and all 2s in last.
So the output will be 1,1,0,0,2,2

Constraint
The number list must have only 0, 1 and 2
Size of list A<=100

Input
8
1 0 2 0 1 0 1 2

Output
1 1 1 0 0 0 2 2
